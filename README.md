# Atividade de Lógica com Operadores

## 🧠 Sobre a Atividade

Esta atividade é a continuação das anteriores e tem como objetivo aprofundar o entendimento sobre **operações lógicas**, utilizando três operadores fundamentais que permitem trabalhar com múltiplas condições em tomadas de decisão.

A atividade foi desenvolvida **em duplas**, incentivando a colaboração, troca de ideias e resolução conjunta de problemas.

## 🕵️‍♂️ Os Três Agentes da Lógica

Exploramos três operadores lógicos principais:

* **E (Conjunção):** verdadeiro apenas se todas as condições forem verdadeiras.
* **OU (Disjunção):** verdadeiro se pelo menos uma condição for verdadeira.
* **NÃO (Negação):** inverte o valor lógico (verdadeiro vira falso e vice-versa).

## 🎯 Objetivo

Compreender como combinar diferentes condições lógicas para construir decisões mais complexas em algoritmos e programas.

## 🤝 Metodologia

* Atividade realizada em duplas
* Discussão e análise conjunta dos resultados
* Aplicação prática dos operadores em exercícios

## 🚀 Conclusão

O uso dos operadores lógicos permite criar estruturas de decisão mais completas e eficientes, sendo essencial para o desenvolvimento de programas e resolução de problemas computacionais.
